how it works

One pipeline.
Three layers.

crash-sym auto-detects your crash type and runs all symbolicators in parallel — iOS native, Android native, and JS — then merges everything into one clean report.

01
Parse
Reads your raw crash log. Detects iOS, Android, RN, or ANR automatically from content patterns.
→
02
Detect
Scores the log against platform pattern specs. Identifies architecture (arm64, x86_64) and thread count.
→
03
Symbolicate
Runs dSYM via atos/llvm-symbolizer, ProGuard retrace, and mozilla/source-map — all in parallel.
→
04
Report
Outputs a merged report as text, rich HTML with progress bar, or machine-readable JSON for CI.
platforms

Every mobile platform.
One tool.

Your symbols stay on your machine. No upload. No account. Works on macOS, Linux, and Windows CI.

iOS

Symbolicates Apple crash reports (.crash, .ips) using your dSYM bundle. Uses atos on macOS and llvm-symbolizer on Linux/Windows.

dSYM atos llvm-symbolizer arm64
Android

Deobfuscates ProGuard/R8-minified stack traces using your mapping.txt. Handles nested exceptions and Caused by chains.

ProGuard R8 mapping.txt ANR
React Native

Maps minified JS bundle errors back to your original TypeScript/JavaScript source using source maps from Metro or Expo.

source-map Metro Expo TypeScript
usage

Simple commands.
Powerful output.

Auto-detect platform or specify it. Text, HTML, or JSON output. CI-ready.

terminal 
# React Native — auto-detect + HTML report
$ crash-sym symbolicate crash.log \
  --dsym ./MyApp.dSYM \
  --mapping ./mapping.txt \
  --sourcemap ./dist/index.map \
  --format html --output report.html

→ Detected platform: RN
→ Parsed 1 thread, 7 frames
✓ Report written to: report.html
✓ 7/7 frames symbolicated

# iOS only
$ crash-sym ios crash.log --dsym ./MyApp.dSYM

# Android only
$ crash-sym android crash.log --mapping ./mapping.txt

# JSON output for CI pipelines
$ crash-sym symbolicate crash.log --format json \
  --output report.json
text

Terminal-friendly output with frame counts, thread labels, and symbolication progress.

html

Rich report with platform badges, progress bar, layer coloring, and shareable file.

json

Full structured output for CI pipelines, custom dashboards, and tooling integrations.
